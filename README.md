Projeto AI-102 – Chat com Reconhecimento e Síntese de Fala
Este projeto é um chat inteligente desenvolvido em Python, integrado com recursos de reconhecimento de fala e síntese de voz, permitindo que o usuário interaja de duas formas:
Falando pelo microfone, ou
Digitando normalmente
O sistema processa a entrada do usuário, interpreta a mensagem e gera uma resposta em texto e/ou áudio, criando uma experiência de interação mais natural utilizando Inteligência Artificial.

 Funcionalidades
Reconhecimento de fala (converte áudio em texto)
Síntese de fala (converte texto em áudio)
Interação completa por voz ou texto
Chat funcional utilizando modelo de IA
Processamento de dados em tempo real
Arquitetura simples e organizada em pastas

 Estrutura do Projeto
O arquivo principal do projeto está localizado em:
Reconhecimento_OpenAI/
 Chat_Completo/
   Chat.py   ← Arquivo principal

 Como Executar
Abra a pasta do projeto.
Navegue até:
Reconhecimento_OpenAI/Chat_Completo/Chat.py
Execute o arquivo Chat.py normalmente.
O sistema funciona com microfone (para fala) ou teclado (para texto), sem necessidade de instalar bibliotecas adicionais.

 Objetivo
Este projeto foi desenvolvido como parte dos estudos para a certificação AI-102, com foco em aplicações práticas de Inteligência Artificial e no uso de voz como forma de interação humana-computador.
